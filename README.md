# API Weather App Explanation
The weather app that I have developed allows users to input any city of their choice and the program will return back the temperature, humidity and a description of the weather. It also allows users to view the history of the cities they have searched up and also allows users to generate themselves a graph which shows them the city and the temperature of the city. The program also includes a choice for the user to exit the program. 

# How to run the API
To properly run this API, you need to downaload requests, pandas and matplotlib. Requests is the one which will retrieve data on the inputted city, pandas will save the history of the users inputs and matplotlib is the one which will graph the temperature graph. To install these, the user must go to the temrinal and type the following: pip install requests, pip install pandas and pip install matplotlib. These are all the required packages required for the user to run the code smoothly.
